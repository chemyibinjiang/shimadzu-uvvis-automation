# 吸光度空白与基线校正

## 结论

常规溶液吸光度测量应使用经过空白基线校正的 LabSolutions `Absorbance` 结果，不应在上层程序中
再用导出的吸光度手工减“暗电流”和“空气能量”。自动控制手册公开的是基线校正/自动调零命令
`Command=21`，没有公开独立读取或扣除暗电流的文本交换命令。

这并不证明仪器内部没有暗信号处理，只表示该处理属于仪器和 LabSolutions 的测光流程，不是本
项目可以用 `SPC_CMD.txt` 分别控制的原始信号运算。如果实验必须审计原始暗信号、参考光束能量
或灯能量，应另行建立 Energy 模式验证和厂商确认流程，不能从标准吸光度文件反推。

## Command 21

加载测量方法后，可以发送：

```text
Command=21
CorrectionType=1
```

`CorrectionType=1` 表示自动选择校正方式。对于 Spectrum，它会在已加载 `.vspm` 的扫描范围内
执行基线校正。对于只有一个固定波长的 Photometric、Quantitation 或 Time Course 方法，它通常
在该波长执行自动调零；多波长或范围方法则按手册规则执行范围基线校正。

也可以显式指定 Spectrum 基线范围：

```text
Command=21
CorrectionType=2
StartWL=400.0
EndWL=700.0
```

单波长自动调零为：

```text
Command=21
CorrectionType=3
WL=520.0
```

`CorrectionType=3` 只校正一个波长，不能代替 `400-700 nm` 完整光谱的范围基线校正。命令省略
`CorrectionType` 时手册规定按 `1` 处理，但本项目必须显式发送，避免隐含默认值。

## 溶液样品的放置

对于一套样品位和参比位，推荐使用配对比色皿，并按以下顺序操作：

1. 仪器预热并稳定，连接 LabSolutions，加载本批次将使用的方法。
2. 样品位和参比位都放入与样品基质一致的空白溶液，例如相同溶剂、试剂和 pH 条件。
3. 操作人员确认空白身份、比色皿方向、液面、气泡和外壁清洁状态。
4. 发送 `Command=21, CorrectionType=1`，等待反馈 `Return=0`。
5. 保持参比位不变，只把样品位的空白换成待测样品。
6. 设置该样品的 SampleID 和数据文件，再发送测量命令。

对于液体样品，空气通常不是合适的实验空白，因为空气基线不能消除溶剂、试剂和比色皿的吸收
及散射。只有方法和实验 SOP 明确规定以空气为空白时，才使用空气校正。

## 多样品批次

同一方法、同一空白、同一参比比色皿且仪器状态稳定时，通常在批次开始前执行一次基线校正，
然后逐个更换样品位中的样品。后续样品直接复用该基线，不增加基线确认步骤。以下情况应由上层
结束当前批次并要求新批次重新校正：

- 更换方法、扫描范围、信号类型或关键测量参数；
- 更换空白溶液、参比比色皿或比色皿方向；
- 仪器重新启动、光源状态改变或发生明显漂移；
- 空白复测超出实验室 SOP 的允许范围；
- 批次持续时间超过实验室规定的基线有效期。

长批次应在预定间隔插入空白 QC，而不是假定一次校正永久有效。每个批次的 manifest 应记录方法
文件及哈希、校正类型、校正波长范围、空白 ID、校正时间、反馈 Return 值和操作人员确认信息。

## 自动控制顺序

Spectrum 批次推荐顺序为：

```text
Command=0                         Hello
Command=1                         需要时连接仪器
Command=100                       加载 .vspm
人工门禁                          放置并确认空白
Command=21 CorrectionType=1       执行方法范围基线校正
人工门禁                          将第一个样品放入样品位
Command=110                       设置样品和数据文件
Command=111                       测量
人工门禁                          更换下一个样品
Command=110 / Command=111         逐个测量
```

校正是物理仪器动作。规划工具可以返回校正计划，但执行器在收到空白放置确认前不得发送
`Command=21`。
